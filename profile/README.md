# podgarr

Self-hosted infrastructure, media automation, and developer tooling — built for the homelab and beyond.

## 🔧 What We Build

**podgarr** publishes open tools across a few interconnected domains:

- **Media management** — orchestration, EPG ingest, XMLTV, and *arr-stack automation
- **VPN & networking infrastructure** — Gluetun sidecars, IPTV proxying, and homelab networking primitives
- **OpenAPI specs & developer tooling** — high-quality, empirically validated API specs for tools that deserve better documentation
- **LLM & agent workflows** — architecture patterns for stable multi-model collaboration
